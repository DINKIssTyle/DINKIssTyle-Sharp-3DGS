- 최종적으로 단일 이미지를 이용 3DGS 뷰어를 이용하고 3D 포맷 등으로 출력한다. 이 프로그램은 Go로 제작해서 Mac, Windows, Ubuntu에서 구동된다.

1단계 구현 목표
- https://github.com/apple/ml-sharp (ml-sharp 폴더) 의 sharp 기능을 이용하여 단일 이미지를 3D gaussian-splatting (이하 3DGS) .ply 파일로 변환한다.
- 생성된 .ply 파일 또는 불러온 .ply 파일을 https://github.com/ArthurBrussee/brush (brush 폴더) 의 3DGS 뷰어 기능을 이용해서 3D 뷰어로 확인한다.
- 이 과정에서 사용자가 쉽게 구동준비를 할 수 있는 단계나 프로세서를 간소화하거나 자동화 한다.

2단계 구현 목표
- .obj, .fbx, glb 파일등으로 출력한다.

- 각 라이센스에 준수하는지도 확인되어야 합니다.


https://github.com/ArthurBrussee/brush 가 아이파치 2.0 라이센스이니,
맥용 바이러리를 사전 빌드해서 탑재하고, 이 프로그램은 이제 멀티 플랫폼이 아닌 swift 로 개발합니다.
다만 https://github.com/ArthurBrussee/brush 에서 뷰어 기능만 가져다가 쓰면 좋겠습니다.
